Angular Language Service

Angular Language Service, kod editörleri için geliştirilmiş bir eklentidir. Angular şablonları içinde tamamlamalar için yardımcı olur, hataları detaylı olarak götsterir, ipuçları verir ve referanslara yönlendirir. Angular Language Service, Visual Studio Code, WebStorm ve Sublime Text için eklenti olarak indirilebilir.
