Structural Directives

Structural directive'ler HTML elemanlarına eklenirler. Eklendiği elemanın içeriği değiştirebilir, yeni eklemeler yapabilir veya DOM'dan kaldırabilirler.

  • {{ i + 1 }}. {{ hero }}

*ngFor

  • {{ i + 1 }}. {{ hero }}
